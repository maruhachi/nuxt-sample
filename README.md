# nuxt_sample

> Nuxt.js suburi project

## Build Setup

``` bash
# install dependencies
$ yarn

# serve with hot reload at localhost:3000
$ yarn dev

# build for production and launch server
$ npm run build
$ npm start

# generate static project
$ npm run generate
```
